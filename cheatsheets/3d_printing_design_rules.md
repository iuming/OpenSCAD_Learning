# 3D 打印设计规则速查

本页把 OpenSCAD 建模和 FDM 打印经验连接起来，适合设计盒子、机构、夹具和可动件时快速查阅。

## 1. 壁厚、圆角与强度

- **壁厚**：喷嘴 0.4 mm 时，常用壁厚为 `1.2 / 1.6 / 2.0 mm`（3、4、5 道线宽）。
- **底厚**：承重件建议 `2.0–3.0 mm`；装饰件可 `1.2–1.6 mm`。
- **圆角**：内角加圆角可降低应力集中；外角圆角更耐摔，也更不刮手。
- **加强筋**：筋厚约为壁厚的 `40%–70%`，过厚会缩水。

```scad
// 简单加强筋：用 hull() 连接两个薄柱，形成三角支撑
module rib(length=30, height=15, thick=1.2) {
    rotate([90,0,0])
        linear_extrude(height=thick, center=true)
            polygon([[0,0], [length,0], [0,height]]);
}
```

## 2. 悬垂与桥接

| 结构 | 经验规则 | 设计建议 |
|---|---|---|
| 悬垂 | 45° 内通常安全 | 大角度悬垂改成倒角或拱形 |
| 桥接 | 10–20 mm 内较容易 | 长桥加中间支撑柱或改方向 |
| 圆孔水平打印 | 顶部容易下垂 | 用泪滴孔（teardrop hole）替代圆孔 |

```scad
// 水平泪滴孔截面，适合减小顶部支撑需求
module teardrop_2d(r=5, angle=45) {
    union() {
        circle(r=r);
        polygon([[0,0], [r*cos(90-angle), r*sin(90-angle)], [-r*cos(90-angle), r*sin(90-angle)]]);
    }
}
```

## 3. 孔、柱、螺丝与热熔螺母

- M3 通孔：`3.2–3.4 mm`。
- M3 自攻塑料孔：先试 `2.6–2.9 mm`。
- M3 热熔螺母孔：按螺母外径试，一般 `4.0–4.6 mm`，深度略大于螺母高度。
- 螺丝柱外径：通常为孔径的 `2.0–2.5 倍`，周围加筋更可靠。

```scad
module screw_boss(h=10, outer_d=8, hole_d=3.0) {
    difference() {
        cylinder(d=outer_d, h=h);
        translate([0,0,-0.1]) cylinder(d=hole_d, h=h+0.2);
    }
}
```

## 4. 配合测试优先

不同打印机、材料、速度、温度都会改变尺寸。正式打印前先做小测试片：

```scad
// 生成 0.1 到 0.6 mm 的间隙测试孔
for (i=[0:5]) {
    gap = 0.1 + i*0.1;
    translate([i*16,0,0])
        difference() {
            cube([14,14,4], center=true);
            cylinder(d=6+2*gap, h=6, center=true);
        }
}
```

## 5. 面向切片的建模习惯

1. 模型底面放在 `Z=0`，方便切片软件摆放。
2. 避免极薄悬空片，薄片厚度至少接近喷嘴线宽。
3. 差集切削体多伸出 `0.2 mm`，减少共面错误。
4. 圆柱孔使用 `$fn = 32/48/64`，过高只会拖慢渲染。
5. 可动件加 `clearance` 参数，默认不要写死。

## 6. 导出前检查清单

- [ ] 有无不可打印的悬垂或长桥？
- [ ] 孔径是否包含打印公差？
- [ ] 可动件是否留足间隙？
- [ ] 是否避免非流形：共面、零厚度、自相交？
- [ ] 是否用小样件验证材料与打印机？
